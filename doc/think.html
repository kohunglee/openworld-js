Current API Closure

当前真正闭环的是 signboard 的 boards 协议。它已经可以支撑内容空间化 MVP。

GET /api/signs

全量读取信息板，供热点侧栏和 admin 管理页使用。

POST /api/signs/batch

按 ID 合并懒加载，让大量墙面不必入口全量请求。

PATCH /api/signs/:id

3D 编辑面板单条 upsert，保存文字、图片 URL 和备注。

GET /api/signs/stream

SSE 实时推送变化，让管理后台、AI、3D 客户端联动。

POST /api/signs

admin 批量替换保存，适合导入器和迁移工具谨慎使用。

8-Week Plan

02

Fixed villa template

入口、走廊、3 个房间、全文模态框、图片板，保证录屏好看。

03

SaaS project page

项目、2D 列表、打开 3D、免费限制、公开链接，不做重 OAuth。

04

Market signal

英文 landing page、录屏、20-50 个海外用户触达、$99-$499 setup。

Visual Proof Assets

当前仓库里已有可用于样板页和演示的视觉素材。后续真正销售页应换成录屏和真实 3D 空间截图。

What to avoid in 8 weeks

  • 不要做完整多用户 OAuth，先用 Markdown/Lark/手工导入验证价值。
  • 不要重构生产 signboard server，先用现有 boards API 跑通样板。
  • 不要做无限建筑生成，先做一个固定但可录屏的 villa 模板。
  • 不要先追求订阅自动化，第一笔钱优先靠 done-for-you setup。
