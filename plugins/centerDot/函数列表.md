## 按功能分类索引
==================

### 编辑器相关
```
initHTML()              - 插入编辑器
insertEdiFromBackUp()   - 填充编辑区
hotAction()             - 打开编辑器
cancelAction()          - 取消
quitPanel()             - 退出面板
```

### 数据相关
```
getCubesData()          - 获取数据
saveToLocalSt()         - 保存数据
```

### 模型操作
```
displayHotModel()       - 高亮显示
operaCube()             - 复制/添加方块
modelUpdate()           - 更新模型
```

### 键盘鼠标
```
keyEvent()              - 键盘事件
unlockPointer()         - 解锁鼠标
lockPointer()           - 锁定鼠标
```

### 面板
```
panelMoveInit()         - 开始拖拽
panelMove()             - 拖拽中
panelMoveEnd()          - 结束拖拽
onclickView()           - 点击画面
onchangeForeach()       - 输入框事件
```

### 面板按钮
```
e_presets()             - 翻转
e_round()               - 归整
e_zero()                - 归零
e_delete()              - 删除
```

### 基点
```
bassSet()               - 设置基点
deformationBase()       - 基点形变
```

### 参考
```
displayRefer()          - 显示参考方块
```

### 显示
```
showScreenHotInfo()     - 显示热点信息
drawFDico()             - 画箭头
```

### 工具
```
f()                     - 保留小数
nDeg()                  - 规范化角度
music()                 - 播放音效
```
